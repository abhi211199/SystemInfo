# SystemInfo
#### This is a cross-platform desktop app built on `Electron` and `NodeJs`. This app displays the system information like OS config, CPU config etc.
To test the app in your system, please follow the given steps:
- Ensure that the latest version of `npm` and `node` are installed in your system.
- To install `electron` use, `npm install --unsafe-perm=true`
- To start the app, use `npm start`.
